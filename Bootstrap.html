RISTORANTE CON FUSION - BOOTSTRAP

This project deals with the design and development of the front-end UI of an imaginary restaurant called Ristorante Con Fusion using the Bootstrap 4 framework. It also uses some JQuery syntax. The four sections of the website, namely Home, About Us, Menu and Contact were represented by four different HTML files. The code makes extensive use of the style features provided by Bootstrap 4 to augment the basic HTML syntax and make the website visually appealing to the user. There are also modal popups for logging in and making a reservation. The code was run using a local lite server that was set up using Node and NPM. This project was a part of the Coursera course: Front-End Web UI Frameworks and tools: Angular 4 and the assets for the project like images and other information were provided by the course developer.
